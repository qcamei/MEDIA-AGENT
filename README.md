# MEDIA-AGENT
使用各类大模型和AI工具生成和下载文档，图片，音乐，视频，3D
1. 在首页点击"创建媒体"按钮
2. 选择：
   - 主题
   - 媒体类型
3. 点击"生成媒体"按钮
4. 等待媒体生成完成（您可以实时查看进度）
5. 下载或预览生成的媒体
